# **Client application for TimeControl project.**

**Used patterns:**

- Architectural:
  - MVC (Model-View-Controller)
- Behavioral:
  - Observer
- Creational
  - Singleton

**Used technologies:**

- Java
- JavaFX
- Maven

**Additionally:**

- Interaction is implemented through a GUI.
- Business logic of the project is implemented on the server.
